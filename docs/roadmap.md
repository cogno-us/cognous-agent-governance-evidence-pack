# Roadmap

This document outlines planned and aspirational improvements to the Agent Governance Evidence Pack.

---

## Near-term

- **Replay bundle import helper** — a utility to import replay bundle metadata from common replay bundle formats.
- **Action manifest import helper** — a utility to seed a tool and action inventory from an agent action manifest.
- **Evidence pack diffing** — compare two evidence pack versions to identify what changed between review cycles.
- **Richer executive summaries** — enhanced narrative generation for executive and board-level summaries.
- **CSV export** — export risk register, action inventory, and tool inventory as CSV.
- **DOCX export** — export rendered evidence pack as a Word document for distribution.
- **Additional example industries** — evidence pack examples for healthcare, financial services, legal, and HR use cases.
- **Conformance tests** — a formal conformance test suite for validating third-party implementations of the evidence pack schema.

---

## Long-term

- **Hosted evidence review workflow** — a lightweight web interface for reviewing and approving evidence packs.
- **Compliance framework mappings** — mapping evidence pack fields to common compliance control frameworks.
- **Reviewer routing** — workflow support for routing evidence packs to the right reviewers.
- **Enterprise approval workflows** — integration with enterprise identity and workflow systems for approval recording.
- **Dashboard integrations** — connectors for governance dashboards and GRC platforms.
